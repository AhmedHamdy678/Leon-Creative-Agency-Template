**Leon** is a single-page creative agency portfolio template designed with a focus on minimalism and modern aesthetics. It features a clean layout, smooth interactions, and a fully responsive design that works seamlessly across all devices, from desktops to mobile phones.

This project serves as a perfect starting point for agencies, freelancers, or anyone looking to showcase their portfolio in a professional manner.

## âœ¨ Features

- **Responsive Design:** Built with a mobile-first approach, ensuring the site looks great on all screen sizes.
- **Modern Layout:** Utilizes CSS Grid and Flexbox for a flexible and robust structure.
- **Custom Design System:** Implements CSS Variables (Custom Properties) for consistent theming and easy maintainability.
- **Interactive Elements:** Smooth scrolling, hover effects, and modern transitions.
- **Iconography:** Integrated **Font Awesome** for scalable vector icons.
- **Typography:** Uses **Google Fonts** (Work Sans) for excellent readability and style.
- **Clean Code:** Semantic HTML5 markup and organized CSS structure.
